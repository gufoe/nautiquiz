# Nautiquiz

Applicazione per esercitarsi con i quiz della patente nautica. Il dataset base in repo usa l'Allegato A ufficiale MIT 2025 per la categoria D1. Available on [gufoe.it](https://nautiquiz.gufoe.it/)

## Usage

This is a fully client-side Vue application made with Quasar framework.
Start the development server with:

```bash
bun dev
```
